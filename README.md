### Installation

To install the application follow the steps below in the console, you need to have installed nodejs, git and yarn.

Download repo:
```sh
git clone https://github.com/dommirr/starwars.git
```

Change to directory:
```sh
cd starwars
```

Install dependencies:
```sh
yarn install
```

Run App:
```sh
yarn start
```
